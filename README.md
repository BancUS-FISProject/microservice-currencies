# Generated Docker Image

With this command you can pull latest stable version of the microservice.

```commandline
docker pull alvvigsua/microservice-currencies:latest
```

Also, you can pull images from previous pull requests using the merge sha.

```commandline
docker pull alvvigsua/microservice-currencies:<tag>
```

```commandline
https://github.com/BancUS-FISProject/microservice-currencies/tree/<tag>
```
